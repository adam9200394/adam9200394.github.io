muscle memory attirbutes

human depend on muscle memory in a lot of thing on servral activities the muscle memory is one of many functions produced by brain activity it used in pattrens of movement pattrens of speach and langues and many other things this algorthim aim to make very simplifyed mathmentical description of the functionality of the muscle memmory and it do so by the following :

  • - first establishing parameters for the network.
  • - those parametes make the memory.
  • - when an action is made for the first time a pattren emerge.
  • - every time the same action is made the pattren grow stronger.
  • - this action then could be used again either by stimulatus input or a smart network decision.
